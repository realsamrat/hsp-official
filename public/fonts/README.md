# Custom Fonts Directory

This directory contains custom font files for the Headshot Portland website.

## Font Formats

For optimal performance and browser compatibility, fonts should be provided in the following formats:
- .woff2 (Primary format - best compression and performance)
- .woff (Fallback for broader browser support)
- .ttf (Additional fallback if needed)

## Included Fonts

- LTC Bodoni 175 Pro (Regular, Bold)
- Inter (Regular, Medium, SemiBold, Bold)

## Usage

These fonts are loaded via the Next.js Font Optimization API in `app/fonts.ts`.
